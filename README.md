## ✨ 为什么是“情绪调酒”？

在快节奏的生活中，我们的情绪往往是一团乱麻。本项目通过 AI 深度解析用户输入的文字或语音，将其转化为一杯杯具有色彩、层次和味道的虚拟饮品，帮助用户实现**情绪自愈**。

---

## 🚀 核心亮点 (Core Features)

### 1. 🧪 情感炼金术 (Distilling Essence)
* **动态解析：** 输入“今天不是很开心”，系统会自动捕捉情绪的“明亮度”与“能量值”。
* **多维构成：** 将情绪拆解为**基调、中调、前调、尾韵**，并给出专业的口感/体感描述。

### 2. 📖 双重配方系统 (Double Recipes)
* **意象配方 (The Metaphor)：** 充满诗意的文学化描述，赋予情绪仪式感。
* **现实特调 (The Recipe)：** 提供**可操作的真实配方**（如：伯爵茶 + 牛奶 + 薰衣草），让治愈从屏幕延伸到杯中。

### 3. 🎵 氛围感沉浸 (Immersive Experience)
* **情绪音乐：** 自动匹配与当下心情契合的 Ambient 背景音乐，营造完整的治愈空间。

### 4. 📈 情感旅程 (Mood Journey)
* **酒单收藏：** 记录每一次的心情特调，像收藏名酒一样珍藏过往。
* **洞察与趋势：** 自动生成**周度心语**，多维度分析近期情绪波动。
